--- layout: docs permalink: /docs/panels/revisions-panel/ title: Panel “Revisions” excerpt: >- Lists revision history of selected registry item in selected language. ---

The Revisions panel shows revision history (lineage) for selected registry item in selected language.

Each revision is identified by a hexadecimal code and a timestamp.

The selected revision affects data shown elsewhere.

In context of Edit and Translate modules, the latest revision of currrent item must be selected for changes to be made (building changes on top of obsolete revisions is not supported).

In Review module context, selected revision affects the target that the proposed change will be compared with.
