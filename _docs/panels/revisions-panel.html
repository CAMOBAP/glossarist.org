--- layout: docs permalink: /docs/panels/revisions-panel/ title: The “Revisions” panel excerpt: >- Lists revisions of the currently selected registry item. Selected revision determines which version to show, and controls the “Selected revision” panel. ---

The Revisions panel determines currently selected revision.

Selected revision determines the version of currently selected registry item data shown in different modules.

In Edit/Translate modules, the latest revision of currrent item must be selected (building changes on top of obsolete revisions is not supported).
